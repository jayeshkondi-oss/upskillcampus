# 🌾 Agriculture Crop Yield Prediction

## 📌 Project Overview
This project predicts agricultural crop yield in India using Machine Learning.  
It uses a Random Forest Regressor to estimate yield based on crop type, state, and cost factors.

---

## 🚀 Features
- Predict crop yield instantly
- User-friendly Streamlit interface
- Data visualization (crop-wise & state-wise)
- Real-time ML predictions

---

## 🧠 Machine Learning Model
- Algorithm: Random Forest Regressor
- Metrics Used:
  - R² Score
  - MAE (Mean Absolute Error)
  - RMSE (Root Mean Square Error)

---

## 📊 Input Parameters
- Crop
- State
- Cost of Cultivation (A2+FL)
- Cost of Cultivation (C2)
- Cost of Production (C2)
